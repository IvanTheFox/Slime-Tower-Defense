# Slime Tower Defense
## Краткое описание
Жанр: Tower Defense/Action-RPG;
Сеттинг: Фэнтези;
Цель игрока: Защитить ключевой объект (кристалл) - неподвижную цель в центре карты
### Игровой процесс
Карта разделена на клетки, имеющие свои координаты
#### Базовые механики
 - Управление: WASD/стрелки для движения, стрельба по клику мыши в направлении курсора;
 - Система уклонения (Shift): рывок/перекат, который дает на 1 секунду неуязвимость игроку. Имеет перезарядку ~4 секунды;
 - У игрока есть некоторое количество здоровья. 
 - Если игрок умирает, через несколько секунд он возрождается рядом с Кристаллом и теряет процент от накопленной суммы монет.
 - С каждой смертью время возрождения и потери монет увеличиваются.
 - Здоровье игрока медленно восстанавливается, когда он не получает урона и не атакует 
#### Волны
- Между волнами появляется временный магазин, где игрок тратит заработанные ресурсы; отображается таймер до начала следующей волны;
- Во время волны с двух случайных направлений появляются противники; отображается таймер до конца волны.
- Игрок пытается уничтожить всех врагов волны, не дав им разрушить защищаемый объект.
- Цикл повторяется с более сложной волной, направления атаки меняются
- Точки спавна врагов расположены около границ карты
#### Временный магазин
- Доступен только между волнами
- Исчезает в начале следующей волны
- Игрок может потратить монеты, чтобы купить:
	1. Более сильное оружие случайного типа;
	2. Предмет с особым свойством;
	3. Ловушки случайного типа;
	4. Восстановление ~30% прочности кристалла;
	5. Увеличение прочности кристалла.
- Товары обновляются после каждой волны
#### Ловушки
Игрок может ставить ловушки, чтобы было легче отбивать волны.
- В одну клетку можно разместить только одну ловушку
- Ловушки имеют ограниченное количество срабатываний
- Ловушка активируется, если противник находится на одной клетке с ней
- Ловушки исчезают, если потрачены все использования
- Есть несколько типов ловушек:
	1. Замедляющая, наносящая небольшой урон каждую секунду, нет задержки срабатывания (имеет очень много использований ~200)
	2. Наносящая средний урон раз в 2 секунды - задержка срабатывания - 1 сек, перезарядка - 2 сек (имеет много использований ~80)
	3. Замораживающая, не наносящая урона ловушка - задержка срабатывания - 1 сек, время заморозки - ~3 сек, время перезарядки - ~7 сек (имеет малое количество использований ~40)
	4. Обезоруживающая, не наносящая урона ловушка - задержка срабатывания - 1 сек, противник не способен атаковать ~5 сек, время перезарядки - 10 сек (имеет среднее количество использований ~60)
	5. Обездвиживающая ловушка, наносит небольшой урон - задержка срабатывания - 1 сек, противник не способен двигаться ~5 сек, время перезарядки - ~10 сек (имеет малое количество использований ~40)
#### Противники
Противники пусть будут слаймами, чтобы не заморачиваться со спрайтами
- Зеленый: Мало HP, маленький урон
- Синий: стандартный враг, средние HP, средний урон.
- Фиолетовый: танк - много HP, медленный, наносит большой урон вблизи.
- Красный: атакует с дистанции, мало HP; приоритетная цель - игрок.
- Желтый: Быстрый, когда достигает цели - взрывается, нанося большой урон по площади; цель - Кристалл
- Босс: Появляется каждые ~5 волн. Обладает огромным запасом здоровья, уникальными атаками и механиками. Замедления от ловушек на босса действуют в 2-3 раза меньше по времени
#### Предметы
Предметы позволяют увеличить характеристики игрока, такие как урон, скорострельность, скорость передвижения
Предметы можно купить во временном магазине
#### Стены
Стены позволяют облегчить защиту. Я предлагаю создать несколько карт с готовыми не разрушаемыми стенами
- Стены представляют собой "закрашенные" клетки, через которые не могут пройти противники или игрок
- Через стены нельзя стрелять
- Враги пытаются обойти стены
- Стены являются не разрушаемыми
- На клетке со стеной нельзя разместить ловушку
#### Экономика и ресурсы
Основная валюта - монеты
Источники дохода:
- Уничтожение врагов.
- Бонус за завершение волны без урона объекту.
Расходы - покупка товаров во временном магазине
#### Опциональные механики
- Предметы, позволяющие игроку нанести критический урон
- Ограниченные по времени задания, за выполнение которых игрок получает награду (например, убить 10 врагов за 15 секунд)
